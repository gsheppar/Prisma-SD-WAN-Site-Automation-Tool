Steps To Setup Auth Token

  1. Login to the Prisma SD-WAN Portal
  2. Select User Administration and click the pencil to edit your account
  3. On your profile disable "IP SESSION LOCK" (botton left image)
  4. If you are a Palo Alto Network Employee using your corporate account you must add "0.0.0.0/0" in IP/PREFIX ACCESS LIST
  5. Select System Administration at the top right
  6. Select AUTH TOKENS and generate a token (bottom right image)
  7. Insert your AUTH TOKEN here
